# Employee Management System

This is a full-stack web application built using Spring Boot and React.

## 🔧 Technologies Used
- Spring Boot
- React JS
- H2 Database
- REST APIs

## 🚀 Features
- Add Employee
- View Employees
- Update Employee
- Delete Employee

## ▶️ How to Run

### Backend
cd springboot-backend  
mvn spring-boot:run  

### Frontend
cd react-frontend  
npm install  
npm start
